ME 507 Robot Arm Repository

By: Evan Tran, Lucas Kaemmerer

ME 507 - Charlie Refvem

California Polytechnic State University, San Luis Obispo

Overview

This is the codebase for the mircocontroller code, drivers, vision control, and logic that the robot arm uses to operate. The arm runs using 2 NEMA 17 stepper motors and a NEMA 23, all driven by 3 TMC2240 driver + motion controller chips. The microcontroller is a STM32L4A6ZG (144 pin).
